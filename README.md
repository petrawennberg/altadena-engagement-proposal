# Altadena engagement proposal — CAPS deck

Working group deck proposing a structured community engagement process for the Altadena rebuild. Presented to the CAPS Roundtable, May 21, 2026.

**Collaboration:** Altadena Village · CoLab · Alting · Altagether

## Files

| File | Purpose |
|---|---|
| `deck.html` | **The deck.** Open in any browser. Use arrow keys (← →) to navigate, **F** for fullscreen, **P** to print. The visual source of truth. |
| `deck.pptx` | PowerPoint export — drag into Google Drive and "Open with Google Slides" for collaborative text editing. Lower visual fidelity than the HTML. |
| `deck-spine-v1.md` | Working spine in markdown — speaker notes, structure, and the underlying narrative. Useful for review without rendering the HTML. |
| `process-diagram.svg` | Standalone SVG of the process flow diagram (referenced by `deck.html`). |
| `process-diagram-preview.html` | Standalone preview of just the SVG. |
| `build_pptx.py` | Script that generates `deck.pptx` from the spine content. |
| `braindumps/` | Petra's raw voice notes from May 15 — the origin material for this proposal. |

## How to view the deck

1. Clone this repo
2. Open `deck.html` in your browser
3. Press **F** for fullscreen, **arrow keys** to navigate
